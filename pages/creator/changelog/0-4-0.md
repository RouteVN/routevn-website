## Initial Release

The first public release of RouteVN Creator.

### Features

- Create and manage projects
- Manage assets
  - Images
  - Sounds
  - Characters and character sprites
  - Transforms
- Manage UI
  - Colors
  - Fonts
  - Typography
  - Layout
- Scene Map
- Scene Editor
- Versions and export web version

### Notes

The application is still under heavy development and with many bugs. The best way to keep in touch with us is via Discord.
